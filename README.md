# Data Exploration and Data Preprocessing

## 📌 Project Overview

This project demonstrates the essential steps involved in data exploration and data preprocessing using Python. It covers techniques that prepare raw data for machine learning and data analysis.

The notebook includes practical examples of data transformation and feature engineering using Pandas and Scikit-learn.

---

## 🎯 Objectives

- Understand the dataset
- Perform data exploration
- Apply data preprocessing techniques
- Prepare data for machine learning models

### Data Preprocessing

#### 1. Normalization
- Min-Max Scaling
- Transforming numerical values into a fixed range

#### 2. Standardization
- Z-score Standardization

- Converting categorical variables into numerical features
- Feature engineering for machine learning
